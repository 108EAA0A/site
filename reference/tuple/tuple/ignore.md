#ignore
```cpp
namespace std {
  const unspecified ignore;

}
```
* unspecified[italic]

##概要

<b>ignoreは、[tie](/reference/tuple/tuple/tie.md)を使用してタプルから値を抽出する際に、「不要な値」をマーキングするためのプレースホルダーである。</b>
<b>使用例は[tie](/reference/tuple/tuple/tie.md)を参照。</b>
