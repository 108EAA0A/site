# vector
* vector[meta header]

`<vector>`ヘッダでは、可変長配列の実装である `vector` コンテナを提供する。

このヘッダでは、以下の標準ヘッダをインクルードする：

- [`<initializer_list>`](initializer_list.md) (C++11)


| 名前 | 説明 | 対応バージョン |
|-----------------------------|----------------------------|-------|
| [`vector`](vector/vector.md) | 可変長配列(class template) | |
